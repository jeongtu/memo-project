# memo-project

## 1. 프로젝트 설명
메모장 CRUD를 구현 한 프로젝트

## 2. 팀원 및 역할
- 김정훈 : Application Tier 담당
- 김솔찬 : Presentation Tier 담당
- 김태훈 : Entities 담당
- 이상우 : Main.java 작성하여, 프로그램 흐름 제어 
- 장성준 : Data Tier 담당

## 3. 프로젝트 진행과정
1. 엔티티 팀원별로 작성
2. 엔티티 비교 및 합치기
3. 프로젝트 틀 만들기 
4. 각 Tier별로 역할 담당하여 분배
5. 담당 파트 완성 및 PR
